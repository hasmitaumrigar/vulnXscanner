How It Works

Step 01

Target Input System

Begin every scan by providing a target. VulnX validates and resolves the input before any packets are sent.

  • Enter a domain name, subdomain, or IP address
  • Automatic URL normalization & scheme detection
  • Real-time input validation with error feedback
  • Choose between Quick Scan or Deep Scan mode
Step 02

DNS Resolution & Subdomain Discovery

VulnX resolves DNS records and enumerates subdomains to map out the target's full attack surface.

  • A, AAAA, CNAME, MX, NS, and TXT record lookup
  • Brute-force subdomain enumeration with 2 400+ word list
  • Wildcard DNS detection to filter false positives
  • 800-thread parallel resolution for maximum speed
Step 03

Port Scanning – Normal & Deep Mode

Identify every open port on the target with configurable scan depth. Normal mode checks the top 1 000 ports; Deep mode goes further.

  • Multi-threaded TCP connect scanning
  • Normal scan: top 1 000 common service ports
  • Deep scan: extended range with custom port lists
  • Real-time WebSocket log streaming to your dashboard
Step 04

Service & Banner Detection

Once a port is found open, VulnX fingerprints the service running behind it to map exact software versions.

  • TCP banner grabbing on every open port
  • Service-name mapping (HTTP, SSH, FTP, MySQL, etc.)
  • Version extraction from response headers
  • Technology stack identification for deeper analysis
Step 05

Multithreading & Performance Engine

Speed matters in security. VulnX runs every module on a high-concurrency thread pool so scans finish fast.

  • Up to 800 concurrent workers in deep scan mode
  • ThreadPoolExecutor for CPU-efficient parallelism
  • Socket.IO real-time progress streaming
  • Adaptive rate-limiting to avoid target overload
Step 06

Vulnerability Detection Modules

A dedicated module library checks for the most critical web vulnerabilities automatically.

  • SQL Injection — error-based & time-based detection
  • XSS — reflected & stored payload testing
  • Directory Bruteforce — 750+ path wordlist with soft-404 filtering
  • Admin Panel Detection — common admin endpoints
  • Exposed .env Files — leaked secrets & credentials
  • Security Headers — CSP, HSTS, X-Frame checks
Step 07

AI-Powered Analysis Engine

After data collection, VulnX feeds results into a Gemini-powered AI engine for intelligent risk scoring and remediation.

  • Automated severity classification (Critical → Low)
  • Contextual remediation recommendations
  • CVSS-aligned risk scoring per finding
  • Natural-language summary for non-technical stakeholders
Step 08

Final Report Generation & Download

Export a professional report containing every finding, recommendation, and metric — ready to share with your team.

  • One-click PDF / JSON report download
  • Scan history stored for future comparison
  • OSINT intelligence included in the report
  • Shareable dashboard link for collaborative review
